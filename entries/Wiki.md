## Installation



1. Clone the repository:

   ``` git clone https://github.com/Elhussin/wiki.git```


2. Navigate to the project directory:

   ```cd django-wiki```
3. Install dependencies:



   ```pip install -r requirements.txt```


5. Run the development server:

  `` python manage.py runserver``

## Usage




### Functionality


#### Index

The index page displays a list of all articles available in the wiki. Users can navigate to specific articles by clicking on their titles.

#### Search

The search functionality allows users to find articles by entering keywords. Relevant articles matching the search query will be displayed.


#### Create New Page


Users can create new articles by clicking on the "Create New Page" button. This opens a form where they can provide a title and content for the new article.

#### Edit


To modify existing articles, users can click on the "Edit" button associated with each article. This opens a form pre-filled with the current content, allowing for easy updates.


#### Random


The "Random" feature takes users to a randomly selected article, providing a serendipitous way to discover content.
